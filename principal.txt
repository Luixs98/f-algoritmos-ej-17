Ciclo repetitivo
Es cuando una o varias lineas de codigo se ejecutan mas de 2 veces, lo cual ayuda ahorrar lineas de codigo.
Facilita el manejo y es facil de comprender.
    1  echo "# f-algoritmos-ej-17" >> README.md
    2  git init
    3  git add README.md
    4  git commit -m "first commit"
    5  git remote add origin https://github.com/Luixs98/f-algoritmos-ej-17.git
    6  git push -u origin master
    7  git commit -m "first commit"
    8  git config --global user.email "wlyfm2015@gmail.com"
    9  git config --global user.name "Luixs98"
